\section{Motivación}
%-------------------------------------------------------------------
\label{cap1:sec:motivacion}

Las personas con discapacidad cognitiva presentan dificultades a la hora de comunicarse con lenguaje natural, por esta razón tienen la necesidad de buscar alternativas que les ayuden a superar estas barreras, para ello se utilizan los pictogramas, que permiten una mejor comprensión del lenguaje.

Para permitir la comunicación entre los interlocutores se utilizan los tableros de comunicación, que a pesar de requerir que todos los participantes conozcan los pictogramas, son muy aceptados por su versatilidad. Se pueden utilizar en cualquier situación, para facilitar la comprensión de los mismos se suelen emplear una serie de formatos de manera habitual, así al ver el formato se entiende el mensaje mas rápido. Algunos de los formatos habituales son las normas, las agendas, los calendarios, \dots

Actualmente, a pesar del avance de las tecnologías en muchos centros aun se trabaja manualmente, es decir, para realizar un tablero se utiliza papel, cartulina, recortables para los pictogramas, pegatinas, etc. Incluso para situaciones en las que se necesita un tablero con urgencia se ven en la necesidad de hacerlo en con papel y bolígrafo, ya que es mas rápido y no tienen una herramienta que sea accesible.

A pesar de la multitud de herramientas existentes destinadas a facilitar la comunicación con personas con discapacidad cognitiva, es difícil encontrar alguna que se ajuste a los formatos que el usuario entiende y sus necesidades, esto provoca que sea el usuario el que se tenga que adaptar a las herramientas y no al revés. 

Por todas las razones anteriormente mencionadas, surge la necesidad de desarrollar una aplicación web que permita a los usuarios generar plantillas y tableros, sin tener que empezar de cero cada vez que quieran utilizar algún formato concreto. Al ser una aplicación web los usuarios pueden acceder a ella desde cualquier dispositivo y en cualquier momento siempre que se tenga conexión a Internet.

Para ello utilizamos distintos formatos, normalmente ya definidos, que el usuario reconoce, así es mas fácil que sepa a que nos estamos refiriendo, sobre que le estamos informado, o pidiendo(normas, agenda, elecciones, etc.).

\section{Objetivos}
%-------------------------------------------------------------------
\label{cap1:sec:objetivos}

Uno de los objetivos principales de este trabajo es desarrollar una aplicación web que permita a los usuarios generar las plantillas y tableros, permitiendo ademas que se puedan reutilizar fácilmente, así agilizar la creación de tableros a partir de una plantilla.

Para que la aplicación tenga el mayor alcance posible y se adapte a las necesidades de los usuarios se va a realizar un diseño centrado en el usuario, para ello se mantendrán reuniones y realizarán evaluaciones con los usuarios finales, con el fin de saber las necesidades que tienen y conocer su opinión acerca de la aplicación y su usabilidad.

% que utilizan normalmente. Una vez que se puedan generar las plantillas, el usuario podrá utilizarlas siempre que quiera. Para ello vamos a investigar con los usuarios cual es la manera de hacerlo que mejor se adapta y les permite generar las plantillas mas fácilmente.

También es importante en este trabajo reflejar los conocimientos adquiridos durante la carrera en un proyecto con un sentido social y que ademas sea de utilidad mas allá de ser un proyecto académico, ademas de aprender a trabajar con tecnologías nuevas.
